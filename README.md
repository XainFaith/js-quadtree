# js-quadtree
 A Javascript implementation of a quad tree.
 
 A quad tree is a spatial partitioning techque that is used to seperate out a space to make collision with static terrain faster.
 
 This implementation choses to place objects into the tree where they fit in order to avoid overlapping objects at the lowest level.
 
 There is a basic example of usage provided and is rendered with PIXI.js
